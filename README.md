## Repository Description
This repository is a demonstration of an integrated AI-assisted coding workflow. It showcases the use of the **Windsurf** (by Codeium) AI coding assistant to generate documentation, manually refine content for clarity and personalization, and manage the entire process through Git version control.

## Git Workflow Steps
Throughout this assignment, I followed a professional Git workflow using the terminal:
1. **Initialize:** Used `git init` to set up the local repository.
2. **Setup:** Created initial files and performed the first commit.
3. **Drafting:** Used Windsurf's AI (Cascade) to generate `assistant_draft.md` and committed the change.
4. **Refining:** Manually edited `final_version.md` to reflect my specific academic interests (Neural Networks and Spark) and committed the update.
5. **Connecting:** Linked the local repository to GitHub using `git remote add` and pushed all changes using `git push`.
